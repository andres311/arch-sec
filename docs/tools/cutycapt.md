## Description

CutyCapt is a cross-platform command-line utility to capture WebKit's rendering of a web page into various formats.

## What It Does

- Captures webpage screenshots
- Supports multiple output formats (PNG, PDF, SVG)
- Renders JavaScript
- Custom viewport sizes
